
# Data Cleaning and Visualization Project

This project focuses on cleaning, processing, and visualizing the Titanic dataset using Python.

## Tools Used

* Python
* Pandas
* Matplotlib
* Google Colab

## Tasks Performed

* Loaded the dataset
* Checked missing values
* Removed duplicate records
* Filled missing values
* Created data visualizations
* Saved the cleaned dataset

## Dataset

Titanic Dataset

Successfully performed data cleaning and visualization to gain insights from the dataset.
